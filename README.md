# rz-blog

Laravel `10.x`（PHP `8.2`）でブログサイトを構築。

## 前提条件

開発環境:

- `MacBook Air (M1, 2020)`
- `macOS Ventura 13.4`
- Docker: `20.10.12`
- Docker Compose: `2.2.3`
- PHP: `8.2`
- MySQL: `5.7`
- nginx: `1.25.1`
- composer: `2.5.8`
- Laravel: `10.x`

Docker環境構築: [docker-laravel](https://github.com/rk-techs/docker-laravel) を参照。
